# Amazon Q Developerの設定ファイル

このディレクトリ配下の`amazonq`ディレクトリを`~/.aws/amazonq`にシンボリックリンクしてください。

## Linux/macOS

```shell
ln -s $(pwd)/amazonq ~/.aws/amazonq
```
