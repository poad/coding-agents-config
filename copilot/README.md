# GitHub Copilot CLIの設定ファイル

このディレクトリ配下の`.copilot`ディレクトリを`~/.copilot`にシンボリックリンクしてください。

## Linux/macOS

```shell
ln -s $(pwd)/.copilot ~/.copilot
```
